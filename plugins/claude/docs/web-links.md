# Viv Web Links

A reference of external URLs for the Viv project. When the user needs a link — to documentation, a registry page, a marketplace listing, or a guide — find it here instead of guessing.

URLs are not clickable in Claude Code's terminal. When sharing a link with the user, offer to open it in their browser: `open <url>` on macOS, `xdg-open <url>` on Linux.
